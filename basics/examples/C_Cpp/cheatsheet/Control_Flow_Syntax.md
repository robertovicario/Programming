# Control Flow Syntax

## IF_ELSE

```cpp
if (condition) {
    statement
}
else {
    statement
}
```

## IF_ELSE_IF

```cpp
if (condition) {
    statement
}
else if (condition) {
    statement
}
else {
    statement
}
```

## WHILE

```cpp
while (condition) {
    statement
}
```

## DO_WHILE

```cpp
do {
    statement
} while (condition);
```

## FOR

```cpp
for (init; condition; assign) {
    statement
}
```

## FOR_EACH

```cpp
for (init : collection) {
    statement
}
```
