# Control Flow Syntax

## IF_ELSE

```cpp
if (/* condition */) {
    /* code */
}
else {
    /* code */
}
```

## IF_ELSE_IF

```cpp
if (/* condition */) {
    /* code */
}
else if (/* condition */) {
    /* code */
}
else {
    /* code */
}
```

## WHILE

```cpp
while (/* condition */) {
    /* code */
}
```

## DO_WHILE

```cpp
do {
    /* code */
} while (/* condition */);
```

## FOR

```cpp
for (size_t i = 0; i < count; i++) {
    /* code */
}
```

## FOR_EACH


```cpp
for (auto &&i : v) {
    /* code */
}
```
